Antes de trabalhar com a API do Telegram, precisa obter o próprio ID de API e hash:

1- Siga este link (https://my.telegram.org) e faça login com o número de telefone.

2- Clique em Ferramentas de desenvolvimento de API.

3- Uma janela "Criar novo" aplicativo será exibida.
Preencha os dados da sua candidatura. Não há necessidade de inserir qualquer URL,
e apenas os dois primeiros campos (Título do aplicativo e Nome abreviado)
podem ser alterados posteriormente.

4- Clique em "Criar aplicativo" no final. Lembre-se de que a hash de API é secreto
e o Telegram não permitirá que você o revogue. Não poste em lugar nenhum!
